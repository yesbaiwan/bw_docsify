# VSCode 文件空包折叠展开

VSCode 空文件夹默认折叠显示，非常不习惯，如图：

![Compact Folders](../../assets/vscode文件夹折叠.png ":size=300")

找到 VSCode 的设置页，在设置页面搜索 `Compact Folders`，把这项勾选掉，就可以了。

![Compact Folders](../../assets/Compact_Folders.png ":size=800")
